exit for

for ... nextループから抜け出て next | end forの次のコマンド文からプログラムの実行を続ける。ゆえに next | end for exit for と next | end forの間のコマンドはスキップされる。
書式:
exit for

sdlBasicの場合は nextの次の行からプログラムを実行を続ける。
例:
' Print numbers from 1 to 10
for i = 1 to 20
    print i
    if i = 10 then : exit for : end if
next
print "done " & i & " iterations"

参照： for ... next
