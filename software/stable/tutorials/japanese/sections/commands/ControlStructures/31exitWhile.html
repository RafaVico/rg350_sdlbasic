exit while

while ... wend 文のloop文の実行をやめて wend | end while 文の位置まで飛び越す。



書式:
exit while

例:
i=0
while i<100
    i+=1
    if i>50 then : exit while : end if  ' -------->     jumps down to the end of the loop...
    prints(i)
end while
