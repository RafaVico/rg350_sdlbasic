for ... next

ブロックコマンドを指定された回数だけ繰り返す。 くり返し制御は変数の値により制御される。

書式:
for variable = start（初期値） to finish（終値） {step increment（増分) }
        commands
        {exit for}
        {continue}
next

制御変数start(初期値）はincrement（増分） によりfinish(終値)まで増加される。
step変数が省略された場合には１だけ増加される。
for と nextの間のコマンドは一度のイタレーションループに一度だけ実行される。
nextの代わりに end forも使用することもできる。

例:

' Print the numbers from 1 to 10
for i = 1 to 10
    print i 
next

参照：exit for, continue ,step
