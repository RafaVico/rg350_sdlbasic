continue

ループの最初にもどる。これはfor...next とwhile...wend のループ内でのみ動作する。

書式:
Continue

下記の例においてcontinue コマンドはループに最初にもどります。そのため'print i ' という文は' i 'という変数が５より大きくなれば実行されません。
例:
'Don't process numbers greater than 5
For i = 1 to 10 ' <----
    If i > 5 Then '       |
                Continue ' -----     this jump to the top of the loop 
            End If
    Print i
End For

参照： for...next ,while...wend
