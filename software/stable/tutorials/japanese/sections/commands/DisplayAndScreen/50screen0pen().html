screenOpen()

論理スクリーンを新たにオープンする。

書式:
screenOpen( num , width , hight , vpx , vpy , vpw , vph , flag )

num スクリーン番号(0-7)
width スクリーンの横幅
hight スクリーンの高さ
vpx ディスプレービュウポートのx座標
vpy ディスプレービュウポートのy座標
vpw ビュウポートの横幅
vph ビュウポートの高さ
flag   透過性の可否
0 = 不透過
2 = 透過+ダブルプレーフィールドがアクティブでない
3 = 透過+ダブルプレーフィールドがアクティブ

例:
screenOpen( 2, 1280, 480, 0, 0, 640, 480, 0 )

例:
screenOpen.sdlbas
