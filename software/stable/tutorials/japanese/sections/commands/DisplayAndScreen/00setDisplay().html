setDisplay()

ディスプレーのパラメーターを変える。
sdlBasicはデフォルトのパラメーターでディスプレーをオープンするが、このコマンドでそれを変えることが出来る。

書式:
setDisplay( width , height , depth , mode )

width ディスプレーの幅
height ディスプレーの高さ
depth ディスプレーのカラービット値.
標準値:
8 = 256色
16 = 65000色
32 = フルカラー
mode ディスプレーモード
標準値:
0 = フルスクリーン
1 = 固定ウィンドウ
2 = リサイズ可能ウィンドウ

例:
setDisplay( 800, 600, 16, 1 )
