screenAlpha()

スクリーンのアルファチャンネル値を設定する。

書式:
screenAlpha( screen , alpha )

alpha 範囲( 0 = 透過 - 255 = 不透過)

例:
circle(320,240,200)
screenopen(1,640,280,0,200,640,280,0)
screenopen(1,640,280,0,200,640,280,0) screenAlpha(1,0)
circle(200,200,200): circle(400,400,200)
for n = 0 to 255
    screenAlpha( 1,n )
