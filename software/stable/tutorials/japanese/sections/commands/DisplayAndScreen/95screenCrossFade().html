screenCrossFade()

現在のイメージから指定のイメージスロットへクロスフェードする。 このコマンドは同期的に動作するのでプログラムを止めない。

書式:
screenCrossFade( screen , imageslot , time )
variable = screenCrossFade( screen )

時間パラメーターなしで呼ばれた場合フェードの状態を返す。
1 動作中
0 終了
Timeの単位は秒数。

例:
circle(320,240,200)
grab(1,0,0,640,480)
cls
circle(200,200,200): circle(400,400,200)
screencCrossFade(0,1,2)
while screenCrossFade( 0 ) =1
    wait(40)
wend
prints( "done! press any key to quit")
waitKey

参照: wait(), screenFadeIn(), screenFadeOut()
