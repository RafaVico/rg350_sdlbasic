screenFadein()

スクリーンを黒から指定のスロットにフェードインさせる。
このコマンドは同期的なので動作中にプログラムを止めない。

書式:
screenFadein( screen , imageslot , time )
variable = screenFadein( screen )

時間パラメーターなしで呼ばれた場合フェードの状態を返す。
1 動作中
0 終了
Timeの単位は秒数。

例:
circle(320,240,200)
grab(1,0,0,640, 480)
cls
screenfadein(0,1,4)
while screenfade( 0 ) =1
    wait(40)
wend
prints( "done! press any key to quit")
waitKey

参照: wait(), screenFadeOut(), screenCrossFade()
