screenFadeOut()

現在の指定のイメージから黒の画面へクロスフェードする。 このコマンドは同期的に動作するのでプログラムを止めない。
旧コマンドのscreenFade() も有効。

書式:
screenFadeOut( screen , time )
variable = screenFadeOut( screen )

時間パラメーターなしで呼ばれた場合フェードの状態を返す。
1 動作中
0 終了
Timeの単位は秒数。

例:
circle(320,240,200)
screenFadeOut( 0, 4 ) ' set up the fade
while screenFadeOut( 0 ) = 1 ' check if fade is still running 
    wait(40)
wend
prints( "done! press any key to quit")
waitKey

参照： wait(), screenFadeIn(), screenCrossFade()
