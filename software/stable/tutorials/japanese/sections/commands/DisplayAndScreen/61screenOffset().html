screenOffset()

現在の論理スクリーン上のディスプレーのビュウポートの位置を設定する。 スクリーンオフセットのコマンドはゲームのプラットフォームで バックグランドのスクロールに役立つ。
offset() コマンドは古いコマンド名だがまだ使用できる。

書式:
screenOffset ( x , y )
offset ( x , y )

例:
screenopen(0,1280,480,0,0,640,480,0)
for i = 0 to 100
    ink(rgb(rnd(255),rnd(255),rnd(255)))
    line(rnd(1280),rnd(480),rnd(1280),rnd(480))
next
x=0
while x<640
    x+=2
    screenOffset( x , 0 )
    waitvbl
end while
