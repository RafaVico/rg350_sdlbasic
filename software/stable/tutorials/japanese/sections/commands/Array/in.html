in

式のダイナミック配列内部にキーが存在する場合にTrueを戻す。 スタティックな配列には使えず、エラーになる。

書式:
expression In array[]
expression is the keyword to find in array array[]

例:
-- Check if key is an index in array a[]
dim a[]
a[1]="one"
a[2]="two"
a[3]="three"
keyword="3"
if keyword in a[] then
    Print keyword; " is a valid index in a[]"
End If
