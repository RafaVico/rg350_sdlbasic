indexes()

配列中のインデックスの数を戻す。

書式:
count = indexes( array[] )

例:
-- Display information about array's indexes
dim a[5,2]
count = indexes( a[] )
Print "There are "; count; " indexes in the array"
for i = 1 To count
    bottom = lBound( a[], i )
    top = uBound( a[], i )
    print "Index "; i; " is from "; bottom; " to "; top
