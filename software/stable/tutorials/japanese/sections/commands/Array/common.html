common

名前をつけてグローバルな配列または変数を作る。

書式:
common variable
common array[]
common name[ maxValue ] [[...]] [, ... ]
common name[ minValue To maxValue ][[...]] [, ...]
common name = expression { , ... }

これらはグローバルにアクセスできる。SubやFunction内で使われるときには特に宣言する必要はない。 同じ結果を得るためにglobalを使うことも出来る。 CommonはSubやFunctionの外側のみで使えるが内側においてはDimを使います。 Dimの使い方に関しては、 Dim項を参照のこと。

例:
' Create an array
common myGlobalArray[3 To 10, 5]

' Create a global variable and assign it
common myGlobalVar = 100
