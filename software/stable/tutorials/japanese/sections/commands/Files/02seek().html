seek()

ファイルのポジションをさがす。ファイルの読み込み/書き込みポイントを指定のポジションに移す。

書式:
seek( stream , position )

stream オープンコマンドで指定された番号
position 読み込み/書き込みするために移動したいポイント。 このポジションはファイルの最初からのバイト単位で移動する。

Example:
open "myfile.txt" for input as 1
seek( 1,2 )
while eof(1)=0
fprints("loc:"+str$(loc(1))+" - ")
file input #1,a$
fprints(a$)
end while
close 1
waitKey
