writeByte()

ファイルに１バイト書き込む。

書式:
writeByte( stream , position )

stream オープンコマンドで指定された番号
position 読み込み/書き込みするために移動したいポイント。 このポジションはファイルの最初からのバイト単位で移動する。

例:
open "myfile.txt" for output as 1
writebyte( 1,asc("s") )
writebyte( 1,asc("d") )
writebyte( 1,asc("l") )
writebyte( 1,asc("B") )
writebyte( 1,asc("a") )
writebyte( 1,asc("s") )
writebyte( 1,asc("i") )
writebyte( 1,asc("c") )
writebyte( 1,13 )
close 1

参照： readByte()
