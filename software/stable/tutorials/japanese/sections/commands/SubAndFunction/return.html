return

function の戻り値をセット。

書式:
return expression

expression は数値,ストリング,変数,配列が可能。

例:
function subtract(a,b)
    return a-b
end sub
print subtract(10,4)

Alternatively you can return function result by assigning it to a variable named as the function.
例:
function subtract(a,b)
    subtract = a-b
end sub
print subtract(10,4)
