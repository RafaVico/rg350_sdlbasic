argument$()

サブルーチンやファンクションの呼び出し元から渡されたアーギュメント。 サブルーチンやファンクションの呼び出し時のパラメーターのやり取りに使用する。 古い argC , argV などの代わりのコマンド

書式:
numberOfArguments = argument$()
argument = argument$( n )

パラメータなしの時は呼び出し元からわたされた アーギュメントの数,パラメーターnが指定されるときｎ番目のアーギュメント。 n
$は省略可 argument()

例: 
sub mySub(...)
for i= 1 to argument
prints( argument( i ) )
next
end sub
mySub("test1")
mySub("test2","more arguments",1,9)
waitkey
