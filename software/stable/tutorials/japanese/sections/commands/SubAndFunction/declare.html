declare

前方参照のための functionName や subNameを作成。 前方参照の Function や Sub を実際のコードを定義する前に参照出来るようになる。


書式:
declare function functionName ( args ... )
declare sub subName ( args ... )

実際にコードが定義されるまで実行は出来ない。 前方参照で問題が起こった場合はOption QBasicを試してみること。

例:
option qBasic
declare function mySub()
' option qBasic causes these statements to be deferred
mySub()
end

function mySub()
    print "This is my function"
end function

参照： option qbasic
