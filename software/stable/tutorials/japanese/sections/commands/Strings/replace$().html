replace$()

文字列sourceを position の位置から置換文字replaceで置き換えて結果の文字列を戻す。
$文字は省略可能 replace() 。

書式:
string = replace$( position , source , replace )
string = replace( position , source , replace )

例:
a$="ciao ciao"
print("string: "+a$)
print( replace( a$,"bye bye" ) )
print( replace( 5, a$,"bye bye" ) )

参照: replaceSubstr$()
