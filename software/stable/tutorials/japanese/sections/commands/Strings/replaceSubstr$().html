replaceSubstr$()

ソースsource文字列の中のターゲットtargetの文字列をreplacingする文字列で置換する。
$文字は省略可能なので replaceSubstr() のように書くこともできる。

書式:
string = replaceSubstr$( source , target , replacing )
string = replaceSubstr( source , target , replacing )

例:
a$="ciao ciao vroby"
prints("string:"+a$)
prints( replaceSubstr( a$,"ciao","bye" ) )
