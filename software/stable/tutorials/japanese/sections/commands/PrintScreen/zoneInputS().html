zoneInputS()

プロンプトを表示してキーボードからユーザーの入力を待つ。 テキストは(x,y)位置に表示される。
 
書式:
variable = zoneInputS( x , y , length , defaultText )

x, yテキストの入力位置
Length 入力を待つ最大文字数、テキストがあふれたら自動的にスクロールされる。
defaultText ユーザーに示されるモデル的なテキスト。

例:
a$ = zoneInputS( 20, 20, 5, "" )
