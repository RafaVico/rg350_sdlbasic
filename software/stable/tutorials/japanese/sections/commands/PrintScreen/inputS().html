inputS()

プロンプトを表示してユーザーからの入力を待つ。 リターンキーが押されるまでの文字列がデータとして入力される。

書式:
variable = inputS( promptString , defaultText )

promptStringはプロンプトとして表示される  
defaultText 単にリターンキーを入力したときに入力されるデフォールトのテキスト。 必要ないときは空のストリング("")を指定すること。  
例:
a$ = ""
while(a$<>"yes")
    a$ = inputS( "do you want to quit?", "no" )
