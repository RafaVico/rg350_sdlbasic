pen()

テキストカラーを設定する。カラーを省略したときは現在のテキストカラーを戻す。

書式:
pen( color )
variable = pen()

color はUint32または１６進法値またはrgb()コマンドで変換した値。 戻り値はUint32フォーマット。

例:
pen( 16711680 )
printS("this is a red text")
printS( "text color = Uint32 " & pen() )
pen( 0x0000ff ) 
printS("this is a blu text")
printS( "text color = Uint32 " & pen() )
pen( rgb(0,255,0) ) 
printS("this is a green text")
printS( "text color = Uint32 " & pen() )
