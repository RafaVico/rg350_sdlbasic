loadImage()

イメージをディスクから指定のイメージスロットに読み込む。 イメージスロットが指定されないときは、フリーなスロットの内の最初のスロットに読み込まれる。 この場合戻り値はこの読み込んだスロットの番号になる。 sdlBasicは65536個のイメージスロットを持つことができる。 LoadImage()コマンドはbmp, gif, iff, jpg, pbm, pcx, png, tga, xpmなどのフォーマットのイメージを読み込むことができる。
 
書式:
loadImage( filename , slot )
slot = loadImage( filename )

例:
s = loadimage( "testimage.png" ) 
pasteicon(0,0,s)
