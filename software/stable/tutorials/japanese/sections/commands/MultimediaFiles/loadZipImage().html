loadZipImage()

zipファイルをディスクから指定のイメージスロットに読み込む。 イメージスロットが指定されないときは、フリーなスロットの内の最初のスロットに読み込まれる。 この場合戻り値はこの読み込んだスロットの番号になる。 sdlBasicは65535個のイメージスロットを持つことができる。 loadZipImage()コマンドは bmp, gif, iff, jpg, pbm, pcx, png, tga, xpmフォーマットのファイルを読み込むことができる。
 
書式:
loadZipImage( zipfile , filename , slot )
slot = loadZipImage( zipfile, filename )

例:
s = loadZipImage( "zipfile.zip", "testimage.png" )
pasteicon(0,0,s)
