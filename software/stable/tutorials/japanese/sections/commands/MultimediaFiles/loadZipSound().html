loadZipSound()

zipで圧縮したサウンドファイルを読み込む。 サウンドファイルはwavフォーマットのみ。 sdlBasicは1024個のサウンドスロットを許可する。 サウンドスロットが指定されないときは、フリーなスロットの内の最初のスロットに読み込まれる。 この場合戻り値はこの読み込んだスロットの番号になる。

書式:
loadZipSound( zipfile , filename , slot )
slot = loadZipSound( zipfile , filename )

例:
s = loadZipSound( "zipfile.zip", "testSound.wav" )
playSound( s, 1 )
waitkey
