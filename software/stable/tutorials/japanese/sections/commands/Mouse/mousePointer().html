mousePointer()

マウスポインターを変更する 通常のマウスポインターはスプライト０、イメージスロット０を使用する。 ポインターの番号を指定しないときは現在の番号を返す。
changeMouse()は古いコマンド推奨しない。
 
書式:
mousePointer( pointer )
variable = mousePointer()

例:
printS("press 0,1,2 or 3 keys")
while not key(k_esc)
    k=chr(inkey)
    if k <> "" then
        mousePointer( k )
    end if
