プログラムの結合

ランタイムのプログラムの結合には２つの方法があります。

自動起動

あなたのプログラムがあるフォルダーの中にsdlBrt.exeをコピーして、 sdlBrt.exeの名前をあなたのプログラムの名前にリネームしてください。
例えばmyprog.sdlbasという名のプログラムがあった場合、sdlBrt.exeをmyprog.exeとりネームします。 そうすることによりmyprogとタイプするとあなたのプログラムがスタートします。

myprog
 
またはアイコンをダブルクリックします。



コードの埋め込み

自動起動はsdlBasicのエディターの中からプログラムコードを埋め込むことでもできます。
エディターのプラグインメニューplugins menuから create executableを選択してください。
あなたのプログラムは同じフォルダーのなかに埋め込み起動のファイルとして保存されます。 リーナックスの場合には、新たに作成されたファイルに起動可能の設定をする必要があります。 （ウィンドウズの場合は必要ありません。） あなたはダブルクリックまたはシェルから呼び出すことによってプログラムをスタートさせることができます。 あなたの埋め込みコードはいかなる保護もされていないことに注意してください。 だれでもが普通のエディターで埋め込まれたコードを抽出することができます。 埋め込みコードはあなたのプロジェクトを配布するための簡便な方法にすぎないのです。
