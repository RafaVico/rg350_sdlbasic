インストール

セットアップ操作の間に、インタプリタ(sdlBrt.exeまたはsdlBrt)はシステムパス(一般に「c：\ウィンドウ\プログラム」または「/opt/sdlBasic/bin」)上のフォルダにインストールされて、さらに、2つの無料のフォントは、デフォルトフォントとして「c：\ウィンドウ\フォント」または「/opt/sdlBasic/share/fonts/ttf」の中にインストールされます。
最新版のsdlBasicのバージョンではデフォルトフォントはインストールの必要はありません。テキスト表示の時だけ必要なだけです。 テキスト表示 commands.
WindowsのセットアップではSDLライブラリを含むdllをインストールします。またGNU/Linuxの上では、SDLライブラリがすでにインストールされていることを必要としています。このようにしてインタプリタはシステムにインストールされます。

プログラムを実行はコンソールの上で下記のようにタイプすることでできます：

$ sdlBasic program.sdlbas

それでプログラムは実行されるでしょう。

sdlBasic IDE中ではツールバーの上のボタンを押すことによりベーシックのプログラムを実行することができます
