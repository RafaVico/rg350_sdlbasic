コメント

sdlBasicではコメントラインのために数種類のオプションを許しています。
下記の例のような、アクセント文字（’）：

' this is a comment

 または、C++スタイルコメントラインを使いたいかもしれません：

// this is a comment

 さらに、Macromedia Directorスタイルコメント：

-- this is a comment

 注
sdlBasicはコメントラインとして使う#文字を許すけれども、これはコード上のまさに最初のラインにあるときだけに有効です。


# first line commenet

[訳注]まだ日本語は使用できません。へたをするとfreezeします。
