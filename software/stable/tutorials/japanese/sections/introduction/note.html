ノート

これはsdlBasic参照ガイドの最初のアルファバージョンであり、まだ不完全で、例題も少なく、いくつかの部分でもっと詳しい説明の必要があります。
sdlBasicの中心的なルーチンではその共有する部分にwxBasicのドキュメンテーションをこのガイドに使用しています。 WxBasicとそのドキュメンテーションはDavid Cunnyによって書かれました。 ほとんどの大部分のコマンド記述は__vroby__によって書かれたオリジナルなsdlBasic のクイックヘルプからのものです。
