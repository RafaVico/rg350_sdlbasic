B2c e' il primo esperimento di scrittura del futuro translatore da sdlbasic a c code.
Non e' ancora completo tuttavia il file cpp generato e' cmq compilabile. 
E' anche possibile scrivere codice basic che puo' essere anche usabile ma il mio punto 
d'arrivo e' spero molto meglio. Nella prossima release sara' implementato un modulo di codice c
chiamato BASengine che fornira' tutte le funzioni di base del basic ma tuttavia non sara' ancora
in grado di compilare il codie sdlbas. Sto anche sinceramente pensando di scorporare 
questo progetto poiche potrebbe anche essere molto interessante usarlo con altre librerie come gtk
qt o altro. Le impressioni e i consigli sono bene accetti.
